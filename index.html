VastuVilla : A all in one RealEstate solution

The VastuVilla website is a best place for both the seller and buyer . In this website the user can easily list their listing on this platform and can easily manage them by creating a listing for renting / selling the property. The user can filter out the listing on the basis of a rent/sell , furnished , parking , no. of bath , no. of bed and price

The user can edit its own listing in easy manner and also can communicate with the person through built in email functionality. The website is optimised for heavy user traffic which leads to give a good user interaction .

Tech Stack used :

  • ReactJs
  • Redux-Toolkit
  • TailwindCSS
  • MongoDB
  • Nodejs
  • Multer
  • Firebase
  • ExpressJs

RealEstate : Best Chatting website

WhisperWave is a feature-rich real-time chat application that enables seamless communication between users. It allows users to sign up, create profiles, chat with individuals via usernames or Gmail, form multi-participant channels, and share files effortlessly. The app is designed with an intuitive UI and robust backend, ensuring a secure and engaging messaging experience.

Tech Stack used :

  • Frontend: ReactJS, ShadCN, TailwindCSS
  • State Management: Zustand
  • Backend: Node.js, Express.js, Socket.io
  • Database: MongoDB
  • Authentication: JWT
  • File Handling: Multer

Features:

  • ✅ User Authentication – Secure login and signup using JWT authentication.
  • ✅ Real-time Messaging – Powered by Socket.io for instant message delivery.
  • ✅ Real-time Messaging – Powered by Socket.io for instant message delivery.
  • ✅ Multi-user Channels – Create and manage group conversations.
  • ✅ File Sharing – Send and receive any file format, including images, documents, and videos.
  • ✅ Search & Invite – Find users easily via username/Gmail and invite them to channels.
  • ✅ Emoji Support – Make chats more engaging with emojis.
  • ✅ Downloadable Files – Users can download received files directly from the chat.
